# CapyCard App

Eine native Desktop-Karteikarten-App, entwickelt mit Avalonia und .NET.

## Fix

- Benennung der PDF-Dateien mit Datum und Uhrzeit beim Exportieren für Eindeutigkeit und Vermeidung von fehlerhaften Dateien ohne .pdf-Endung.
